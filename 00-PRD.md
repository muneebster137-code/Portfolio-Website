## 2. Reference & divergence

Visual mood reference: [Magnetto](https://traditional-screens-722647.framer.app/) — a Framer template for a visual/art-director portfolio. We are **not** cloning it. It's a mood reference for: confident large-type hero, full-bleed project imagery, scroll-driven reveals, an "about → clients → services → milestones" rhythm.

**Where we diverge, and why:**
- Magnetto is built around single moody hero photographs per project. Muneeb's actual work spans wildly different brand palettes across clients (calming pastel for a psychology brand, high-prestige navy for an academic consultancy, dark cinematic for his own studio, earthy minimalism for a flooring brand). A single "moody" visual language would flatten that range — which is the opposite of what should be proven here.
- **Design decision: an editorial/structured grid system as the constant frame, with each case study injecting its own client palette into that frame.** Layout discipline stays constant across all case studies (same grid, same type scale, same component shapes) so the site doesn't feel chaotic — but the color identity visibly shifts per case study to demonstrate range. This is the signature element of the whole site: consistency of structure, diversity of color, used deliberately to prove "I can hold a dozen different brand voices."
- Magnetto's content model (Strategy/Web Design/Branding services, a Journal/blog, a logo-strip "Clients" section) doesn't map onto a personal portfolio for someone without an agency. Cut the Journal entirely. Replace the services section with a skills/competencies section pulled directly from the CV.
